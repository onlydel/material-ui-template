## Toolbar Config

- 툴바를 RichEditor의 스타일이나 플러그인 컨트롤들을 조작하기 위한 컨트롤러 입니다.
- 툴바의 종류는 계속 추가될 수 있습니다.
  - UndoRedo
  - BlockStyle
  - HeadingStyle
  - InlineStyle
  - Image
  - Table
  - CodeDirector
- 툴바를 구분하기 위해 Divider를 추가할 수 있습니다.
